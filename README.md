# Streawman

### About

A self-hosted streaming service for uploading and streaming video. Streams adaptive quality HLS chunks based on the viewer's connection speed. The name is a mix of "streaming" and "straw man". The UI will have a spooky dark Halloween theme.

### AI

This repo heavily uses AI for guidance, review, and quality checks via `.cursor/` and `AGENTS.md`. However, since this is a learning project, all code present in this repo is **MANUALLY WRITTEN ARTISANAL CODE** for the best understanding of the process and learning.
